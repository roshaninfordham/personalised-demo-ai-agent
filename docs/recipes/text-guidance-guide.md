# Text Guidance Guide

Text guidance is the fastest way to improve a URL-only demo.

Include:

- product name;
- target persona;
- top value proposition;
- desired screen order;
- what to avoid;
- tone or talk-track preference.

Example:

```text
Product: Metric Master
Persona: startup founder

Show the dashboard first, then metric creation, then reports.
Focus on weekly revenue metrics and reducing manual reporting.
Avoid billing, payment, delete, invite, account settings, and publishing.
If a capability is not visible, say it was not verified.
```

The generated recipe is a draft. Validate it before using it for a high-stakes demo.
