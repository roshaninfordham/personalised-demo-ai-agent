# Release Checklist

- [ ] Contracts generated.
- [ ] Policies validated and generated.
- [ ] Python lint/type/tests pass.
- [ ] TypeScript lint/type/tests pass.
- [ ] Browser tests pass.
- [ ] Session lifecycle tests pass.
- [ ] Agent evals pass.
- [ ] Docs validate.
- [ ] Docker images build.
- [ ] Security scans pass.
- [ ] Kubernetes manifests render and validate.
- [ ] Staging deploy succeeds.
- [ ] Smoke tests pass.
- [ ] Rollback plan reviewed.
- [ ] No real secrets in repo or artifacts.

Production release requires explicit approval and environment-specific secrets.
